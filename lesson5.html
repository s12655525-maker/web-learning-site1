🎨 Что такое CSS?

CSS (Cascading Style Sheets) — это язык стилей, который используется для описания внешнего вида HTML документов. Если HTML — это скелет страницы, то CSS — это её одежда и макияж.

CSS позволяет управлять:

  • 🎨 Цветами — текста, фона, границ
  • 📝 Шрифтами — размером, типом, стилем
  • 📐 Размерами — шириной, высотой, отступами
  • 🔲 Позиционированием — где элементы находятся
  • 🎭 Анимациями — движением и переходами

📝 Подключение CSS

<!-- Внешний файл (рекомендуется) --> <link rel="stylesheet" href="style.css"> <!-- Внутри HTML --> <style> h1 { color: blue; } </style> <!-- В атрибуте (не рекомендуется) --> <h1 style="color: red;">Заголовок</h1>

🎯 Селекторы

/* По тегу */ h1 { color: blue; } /* По классу */ .highlight { color: red; } /* По ID */ #unique { background: yellow; } /* Групповой */ h1, h2, h3 { font-family: Arial; }

📦 Блочная модель

.box { /* Содержимое */ width: 200px; height: 100px; /* Внутренний отступ */ padding: 20px; /* Граница */ border: 2px solid black; /* Внешний отступ */ margin: 30px; }

🎨 Основные свойства

/* Цвета */ color: red; background-color: #ffffff; border-color: rgba(0, 0, 0, 0.5); /* Шрифты */ font-family: Arial, sans-serif; font-size: 16px; font-weight: bold; /* Текст */ text-align: center; text-decoration: underline; line-height: 1.5; /* Размеры */ width: 100%; height: auto; max-width: 1200px;

📝 Домашнее задание

  1. Подключите CSS файл к HTML
  2. Используйте селекторы по тегу, классу и ID
  3. Настройте цвета и шрифты
  4. Примените блочную модель
  5. Создайте несколько классов с разными стилями
