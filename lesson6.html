Урок 6: CSS Селекторы и свойства

⏱️ 30 минут 📊 Средний 📄 CSS

🎯 Основные селекторы

Селекторы определяют, к каким элементам применяются стили:

/* По тегу */ p { color: black; } /* По классу */ .highlight { background: yellow; } /* По ID */ #unique { font-size: 20px; } /* Групповой селектор */ h1, h2, h3 { color: blue; }

🔗 Комбинаторы

/* Потомок */ div p { color: red; } /* Дочерний */ div > p { color: blue; } /* Сосед */ h1 + p { margin-top: 0; } /* Общий сосед */ h1 ~ p { color: green; }

✨ Псевдоклассы

a:hover { color: red; } /* При наведении */ a:visited { color: purple; } /* Посещённая */ a:active { color: orange; } /* При нажатии */ input:focus { border: 2px solid blue; } li:first-child { font-weight: bold; } li:last-child { color: gray; }

📐 Свойства текста

p { color: #333; font-size: 16px; font-weight: bold; font-style: italic; text-align: center; text-decoration: underline; line-height: 1.5; letter-spacing: 2px; }

📝 Домашнее задание

  1. Создайте страницу с разными селекторами
  2. Используйте классы и ID
  3. Примените псевдоклассы для ссылок
  4. Настройте стили текста
  5. Используйте комбинаторы
