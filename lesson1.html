Урок 1: Введение в веб-разработку

⏱️ 15 минут 📊 Лёгкий 📄 Общее

🌐 Что такое веб-разработка?

Веб-разработка — это процесс создания веб-сайтов и веб-приложений, которые работают в интернете. Это включает в себя написание кода, дизайн, работу с базами данных и многое другое.

Каждый веб-сайт состоит из трех основных технологий:

  • 📄 HTML — структура страницы (скелет)
  • 🎨 CSS — внешний вид (одежда)
  • ⚡ JavaScript — интерактивность (мозг)

🔧 Как работает веб?

  1. Вы вводите адрес сайта в браузере
  2. Браузер отправляет запрос на сервер
  3. Сервер возвращает файлы (HTML, CSS, JavaScript)
  4. Браузер обрабатывает файлы и отображает сайт

💻 Первый HTML файл

<!DOCTYPE html> <html> <head> <title>Моя первая страница</title> </head> <body> <h1>Привет, мир!</h1> <p>Это моя первая веб-страница.</p> </body> </html>

🎯 Зачем учить веб-разработку?

  • 🚀 Высокий спрос на специалистов
  • 💰 Хорошая зарплата
  • 🌍 Возможность работать удаленно
  • 💡 Творческая работа
  • 📈 Постоянное развитие технологий

📝 Домашнее задание

  1. Создайте файл index.html
  2. Откройте его в браузере
  3. Измените текст на странице
  4. Добавьте ещё один заголовок
