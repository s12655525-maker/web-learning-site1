Урок 10: Переменные и типы данных

⏱️ 35 минут 📊 Лёгкий 📄 JavaScript

📦 Объявление переменных

let name = "Иван"; // Изменяемая const age = 25; // Константа var old = "устарело"; // Устаревший способ

🔢 Типы данных

// Примитивные типы let str = "Текст"; // String let num = 42; // Number let bool = true; // Boolean let nothing = null; // null let undef = undefined; // undefined let sym = Symbol("id"); // Symbol let big = 9007199254740991n; // BigInt // Ссылочный тип let obj = { name: "Иван" }; // Object let arr = [1, 2, 3]; // Array

🔄 Преобразование типов

// В строку String(123); // "123" String(true); // "true" 123 + ""; // "123" // В число Number("123"); // 123 Number("abc"); // NaN +"123"; // 123 // В логическое Boolean(0); // false Boolean(1); // true !!"text"; // true

✅ Проверка типов

typeof 42; // "number" typeof "text"; // "string" typeof true; // "boolean" typeof undefined; // "undefined" typeof null; // "object" (историческая особенность) typeof {}; // "object" typeof []; // "object" // Проверка массива Array.isArray([1,2]); // true

  2. Выполните преобразование типов
  3. Проверьте типы переменных
  4. Создайте объект и массив
  5. Используйте const и let правильно
