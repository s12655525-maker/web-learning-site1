Урок 7: Flexbox

⏱️ 40 минут 📊 Средний 📄 CSS

📦 Основы Flexbox

Flexbox — это мощный метод для создания гибких макетов:

.container { display: flex; flex-direction: row; /* row | column */ justify-content: center; /* flex-start | flex-end | center | space-between | space-around */ align-items: center; /* flex-start | flex-end | center | stretch */ gap: 20px; }

🔄 Flex Direction

.flex-row { flex-direction: row; } /* Горизонтально */ .flex-row-reverse { flex-direction: row-reverse; } .flex-column { flex-direction: column; } /* Вертикально */ .flex-column-reverse { flex-direction: column-reverse; }

⚖️ Flex Grow, Shrink, Basis

.item { flex-grow: 1; /* Растягивание */ flex-shrink: 0; /* Сжатие */ flex-basis: auto; /* Начальный размер */ } /* Сокращённая запись */ .item { flex: 1 0 auto; }

📍 Выравнивание элементов

.container { align-items: center; /* Вертикальное */ align-self: flex-start; /* Для отдельного элемента */ } .container { justify-content: space-between; /* Горизонтальное */ }

🔄 Flex Wrap

.container { flex-wrap: wrap; /* nowrap | wrap | wrap-reverse */ align-content: center; /* Выравнивание строк */ }
