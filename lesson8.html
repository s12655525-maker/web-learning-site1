Урок 8: CSS Grid

⏱️ 45 минут 📊 Сложный 📄 CSS

🎯 Основы Grid

CSS Grid — это двумерная система компоновки:

.container { display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-rows: 100px auto 100px; gap: 20px; }

📐 Grid Template

/* Фиксированные и адаптивные колонки */ .grid { grid-template-columns: 200px 1fr 300px; } /* Повторяющиеся колонки */ .grid { grid-template-columns: repeat(3, 1fr); } /* Минимальный размер */ .grid { grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); }

🔲 Grid Areas

.container { display: grid; grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .footer { grid-area: footer; }

📍 Позиционирование элементов

.item { grid-column: 1 / 3; /* От 1 до 3 линии */ grid-row: 1 / 2; } .item { grid-column: span 2; /* Занимает 2 колонки */ grid-row: span 2; /* Занимает 2 строки */ }

⚙️ Выравнивание

.container { justify-items: center; /* Горизонтальное */ align-items: center; /* Вертикальное */ } .item { justify-self: start; /* Для элемента */ align-self: end; }

📝 Домашнее задание

  1. Создайте grid с 3 колонками
  2. Используйте grid-template-areas
  3. Настройте позиционирование элементов
  4. Создайте адаптивный grid
  5. Примените выравнивание
