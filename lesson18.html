Урок 18: Адаптивный дизайн

⏱️ 45 минут 📊 Средний 📄 CSS

📱 Media Queries

/* Минимальная ширина */ @media (min-width: 768px) { .container { width: 750px; } } /* Максимальная ширина */ @media (max-width: 767px) { .container { width: 100%; padding: 15px; } } /* Диапазон */ @media (min-width: 768px) and (max-width: 1024px) { .container { width: 100%; } }

🖥️ Точки останова (Breakpoints)

/* Mobile First */ .container { width: 100%; padding: 15px; } @media (min-width: 576px) { .container { max-width: 540px; } } @media (min-width: 768px) { .container { max-width: 720px; } } @media (min-width: 992px) { .container { max-width: 960px; } } @media (min-width: 1200px) { .container { max-width: 1140px; } }

📐 Относительные единицы

.container { width: 90%; /* Проценты */ max-width: 1200px; margin: 0 auto; } .text { font-size: 1rem; /* Относительно корня */ padding: 1em; /* Относительно элемента */ margin: 2rem 0; } img { max-width: 100%; /* Адаптивные изображения */ height: auto; }

🔄 Адаптивная сетка

.grid { display: grid; grid-template-columns: 1fr; gap: 20px; } @media (min-width: 768px) { .grid { grid-template-columns: repeat(2, 1fr); } } @media (min-width: 1024px) { .grid { grid-template-columns: repeat(3, 1fr); } } /* Автоматическая адаптация */ .grid-auto { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 20px; }

📱 Мобильная навигация

/* Десктоп */ .nav { display: flex; justify-content: space-between; } @media (max-width: 768px) { .nav { flex-direction: column; } .nav-menu { display: none; } .nav-menu.active { display: flex; flex-direction: column; } }

🖼️ Адаптивные изображения

/* HTML */ <picture> <source media="(min-width: 1024px)" srcset="large.jpg"> <source media="(min-width: 768px)" srcset="medium.jpg"> <img src="small.jpg" alt="Адаптивное изображение"> </picture> /* CSS */ img { max-width: 100%; height: auto; object-fit: cover; }

📝 Домашнее задание

  2. Используйте media queries
  3. Примените относительные единицы
  4. Сделайте адаптивную навигацию
  5. Добавьте адаптивные изображения
