⚡ Что такое JavaScript?

JavaScript — это мощный язык программирования, который делает веб-страницы интерактивными. Если HTML — это скелет, а CSS — это одежда, то JavaScript — это мозг сайта.

JavaScript позволяет:

  • 🎮 Реагировать на действия пользователя — клики, ввод текста, прокрутка
  • 🔄 Изменять контент — добавлять, удалять, изменять элементы
  • 🎨 Анимировать — создавать плавные переходы и анимации
  • 💾 Сохранять данные — работать с cookie и localStorage
  • 🌐 Общаться с сервером — отправлять и получать данные

📝 Подключение JavaScript

<!-- Внешний файл (рекомендуется) --> <script src="script.js"></script> <!-- Внутри HTML --> <script> console.log("Привет, мир!"); </script> <!-- В атрибуте (не рекомендуется) --> <button onclick="alert('Привет!')">Нажми</button>

🚀 Первый код

// Комментарий в JavaScript // Вывод в консоль console.log("Привет, мир!"); // Вывод на экран alert("Добро пожаловать!"); // Переменные let name = "Иван"; let age = 25; // Вывод переменных console.log(name); console.log(age);

🎯 Основные типы данных

// Строка let text = "Привет"; // Число let number = 42; // Логическое значение let isTrue = true; // Массив let arr = [1, 2, 3]; // Объект let obj = { name: "Иван", age: 25 };

🔗 Работа с DOM

// Найти элемент const button = document.getElementById("myButton"); // Добавить обработчик событий button.addEventListener("click", function() { alert("Кнопка нажата!"); }); // Изменить текст document.querySelector("h1").textContent = "Новый текст";
